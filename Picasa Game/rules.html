RULES

  • There will be a page for each player that contains his/her name and a box below it
  • Tap the box to see its content
  • All players' boxes, except for one player, have an item in this box (for example: a cat if the category is Animals), the remaining player's box is empty
  • No one is allowed to see the content of any of the other players' box so make sure no one see yours while openning it
  • Then, each player will ask another player a yes-or-no-question about the item and the other player has to answer it
  • The game will determine the player to ask and the player to be asked
  • Based on the answers, the players shall know who is the player that had and empty box
